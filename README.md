# 💰 Personal Finance Tracker

A command-line Personal Finance Tracker built with Python that helps users manage their income, expenses, and monthly budget. The application allows users to record transactions, generate financial reports, and monitor spending while storing all data locally in a JSON file.

---

## 📖 Overview

Managing personal finances is an essential skill, and this project provides a simple yet effective way to keep track of income and expenses. The tracker enables users to add transactions, categorize spending, monitor budgets, and generate summaries through an easy-to-use command-line interface.

This project demonstrates fundamental Python programming concepts including file handling, JSON data storage, modular programming, error handling, and data processing.

---

## ✨ Features

- ➕ Add income and expense transactions
- 📂 Categorize transactions
- 📜 View transaction history
- 📊 Financial summary (Income, Expenses & Balance)
- 💵 Monthly budget management
- 📈 Expense report grouped by category
- 💾 Persistent data storage using JSON
- ✅ Input validation and exception handling

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or newer

### Installation

Clone the repository

```bash
git clone https://github.com/yourusername/personal-finance-tracker.git
```

Navigate into the project

```bash
cd personal-finance-tracker
```

Run the application

```bash
python finance_tracker.py
```
